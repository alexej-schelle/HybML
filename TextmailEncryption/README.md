# 🔐 GAN-Based Email Text Encryption 

---

## 🏫 Project & Repository Description 

📘 This project teaches students the **practical, professional application of methods for analyzing and optimizing electrical networks**. Within a guided seminar with company relevance, students work on **data acquisition, signal analysis, and optimization**. 

The goal of the project is to develop a **Python routine** that allows a **binary signal generated by a circuit** to be restored using a **GAN-based neural network**. The network can also generate possible decimal key values and transform them into **binary sequences**, reconstructing the output signal in binary format. 

This is achieved by generating **binary key values** for a given circuit and **adjusting them using GAN modeling** with respect to a reference signal. Neural networks of this type can be used to **intrinsically encrypt one or multiple decimal numbers as binary sequences**.

## 🇩🇪 Deutsche Version

# 🔐 GAN-Modell zur Verschlüsselung von E-Mail-Textnachrichten 

---

## 🏫 Projekt- & Repository-Beschreibung 

📘 Studierende erlernen die **berufspraktische Anwendung von Methoden zur Analyse und Optimierung elektrischer Netzwerke**. In einem betreuten Praxisseminar mit Unternehmensbezug bearbeiten sie **Datenanalyse, Signalverarbeitung und Optimierung**. 

Ziel des Projekts ist die Entwicklung einer **Python-Routine**, mit der ein **binäres Signal, das durch eine Schaltung erzeugt wurde**, mittels **GAN-basiertem neuronalen Netzwerk** wiederhergestellt werden kann. Außerdem können mögliche **dezimalen Schlüsselwerte generiert** und als **binäre Sequenzen** übertragen werden, sodass das Ausgangssignal im Binärformat wiederhergestellt wird. 

Dies wird erreicht, indem **binäre Schlüsselwerte** für eine gegebene Schaltung erzeugt und **mittels GAN-Modellierung** an den Referenzwert angepasst werden. Solche neuronalen Netzwerke können verwendet werden, um einzelne oder mehrere **dezimalen Zahlenwerte intrinsisch verschlüsselt als Binärzahlen** zu übertragen.
